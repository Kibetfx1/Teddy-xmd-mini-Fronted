# REDXBOT ULTRA — Frontend

Ultra-professional dashboard for REDXBOT ULTRA.

## Deploy to Vercel (Recommended)

1. Upload this `frontend/` folder to GitHub
2. Go to [vercel.com](https://vercel.com) → Import Project
3. Set environment variable: `BACKEND_URL=https://your-backend.up.railway.app`
4. Deploy!

## Or serve locally

```bash
npx serve .
# or
python3 -m http.server 8080
```

Then open `http://localhost:8080`

## Features

- 🔗 Pair your WhatsApp number (no session ID!)
- 📦 Browse all 429+ commands with search & filter
- 📊 Live dashboard with stats
- 🚀 Deploy guide for Railway, Heroku, Render, Docker, VPS
- ⚙️ Backend URL configurator
- 🌙 Ultra dark professional UI
